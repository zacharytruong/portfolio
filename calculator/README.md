# Basic Calculator

## This calculator supports:

1. Mouse
2. Touch
3. Keyboard
4. Numpad

## Basic buttons:

1. Numbers
2. Operators: add, subtract, multiple, divide, and percentage.
3. Clear
4. Delete last input number
5. Period (decimal number)

- HTML

- Responsive UI/UX Design
- DOM Manipulation
- Functions
